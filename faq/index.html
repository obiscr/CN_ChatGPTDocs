    常见问题¶

    1. 如何加入微信群？¶

    回答

    请扫描下方二维码加入微信群

    Tip

    如果二维码已过期，请添加右侧作者的微信，会直接拉入群。

    作者微信

    2. 对网络有什么要求？¶

    回答

    如果是使用API的方式，则可以直接使用。但是如果要是用插件内嵌的浏览器，可能对网络有一定的要求。

    3. ConnectException: Connection timed out: connect?¶

    回答

    这种情况说明你无法访问Open AI的官方服务器，可能需要代理。

    您可以使用以下方法进行测试。使用curl工具直接访问，请记得将以下API Key替换成你自己的： 

    curl https://api.openai.com/v1/chat/completions -H "Authorization: Bearer <Your API Key>" -H "Content-Type: application/json" -d '{ "model": "gpt-3.5-turbo", "messages": [{"role": "system", "content": "Hello!"}] }'

    4. 插件是否可以免费使用¶

    回答

    插件本身是免费的，但是使用官方源的话是要有ChatGPT的账号，这个账号额度可能会需要付费。

    5. 插件在Android Studio中无法使用？¶

    回答

    感谢 @LiLittleCat

    解决方案来自 https://github.com/LiLittleCat/intellij-chatgpt/issues/3#issuecomment-1354157517. 非常感谢 @LiLittleCat

    Step1: Check the ide.browser.jcef.enabled

    首先打开Android Studio，打开任意一个项目。双击 ⇧ Shift 并且搜索: Registry

    确保 ide.browser.jcef.enabled 是选中的

    Step2

    依次打开菜单 Help > Find Action，搜索 Choose Boot Java Runtime for the IDE

    在下拉框选择运行 Select Runtime

    选择一个包含JCEF的Runtime，点击OK会自动下载

    下载完成以后重新启动

    可以看到，现在已经你可以正常使用了。

    如果自动下载失败，可以在 https://github.com/JetBrains/JetBrainsRuntime/releases 此处下载

    然后选择 Add Custom Runtime... 添加

    点击 Add JDK... 选中刚才下载的文件

    如果Runtime出现了任何问题，可以点击 Use Default Runtime 重置。

    6. 为什么我无法在插件市场搜索到这款插件？¶

    回答

    目前插件支持 2020.1 到 2022.3 的版本。如果您的IDE版本小于 2020.1 那么是无法搜索到的。也没有办法使用。请更新IDE到2020.1及其以后的版本。

    7. 遇到问题了如何处理？¶

    回答

    遇到问题时候，可以

    • 在微信群寻求帮助
    • 可以在GitHub 创建一个issue
    • 可以 发送邮件 给作者

    注意

    请注意，不论何种方式，请在描述问题的时候提供更多的信息。例如图片，视频等。

    8. 为什么老是提示 “ChatGPT: 网络读取超时？” 或者 “连接到远程服务器超时？”¶

    回答

    当服务器资源紧张的时候，相应会比较慢。请适当的调整插件设置里面，网络设置的 读取超时 和 连接超时。 比如可以设置到 50000（50秒）。
